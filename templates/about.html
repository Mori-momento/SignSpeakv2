About SignSpeak

SignSpeak is a real-time American Sign Language (ASL) recognition application that uses computer vision and machine learning to interpret hand gestures. The project combines MediaPipe for hand landmark detection with custom-trained SVM and CNN models for gesture classification.

The application features a neumorphic design for an intuitive user experience, with real-time video processing and immediate visual feedback. It's designed to be accessible and user-friendly, helping to bridge communication gaps for the deaf and hard-of-hearing community.
